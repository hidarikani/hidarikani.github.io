## Configuration tips
- Theme - Oceanic Next
- View -> Open View... -> NPM Scripts
- On MacBook Pro with Touch Bar to view function keys when VSCode is focused:
    - System Preferences ->
    - Keyboard ->
    - Shortcuts ->
    - Function Keys -> 
    - Click the Plus Sign , select Visual Studio Code
